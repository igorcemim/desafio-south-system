# Executar via docker
```
docker-compose up
```

# Executar sem docker

## Pré-requisitos
- Utilizar o JDK 8.
- Importar o projeto na IDE utilizando encoding UTF-8.

## Executar a aplicação
```
./gradlew bootRun
```

## Criar um JAR executável
```
./gradlew bootJar
```

## Executar os testes unitários
```
./gradlew test
```
