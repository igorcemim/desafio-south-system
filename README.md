# Pré-requisitos
- Utilizar o JDK 8.
- Importar o projeto na IDE utilizando encoding UTF-8.
- Criar a estrutura de diretórios necessária para a execução do projeto:
  - %HOMEDIR%/data/in
  - %HOMEDIR%/data/out

# Executar a aplicação
```
./gradlew bootRun
```

# Criar um JAR executável
```
./gradlew bootJar
```

# Executar os testes unitários
```
./gradlew test
```
